Adding the jQuery Library

Before you can start coding jQuery, the library needs to be added to your site. It's important to add this script tag BEFORE your other script tags.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.0.0/jquery.min.js"></script>

jQuery Object

$ is a shortcut for jQuery. The $ object is made for selecting an element and then returning that element node to perform an action on it.

The following code selects the element that has the ID header in the HTML.


$('#header');

Document Ready

In order for jQuery to manipulate HTML, it needs to make sure that the HTML is fully loaded before running the jQuery.

So whenever you write jQuery, you'll need to wrap that code in a document.ready function, as shown below:


$(document).ready(function(event){ 
  // jQuery code 
});

Selecting Elements

The great thing about jQuery is how easy it is to select HTML elements. The syntax is pretty simple for selecting an HTML element:

$('CSS CLASS SELECTOR HERE');


$('#footer');           // selects element with footer ID
$('.container');          // selects element with container class
$('ul li');              // selects all li elements within a ul
$('h1'); 				// selects all h1 elements

Each of these selectors are creating a jQuery object of these elements that contains information about the element. Using that jQuery object, you can perform an action on it, such as a click event.

Manipulating HTML Elements

jQuery allows us to easily manipulate HTML elements by reading, adding, or changing attributes or styles of those elements.

Elements can be changed in the HTML, changing their placement, removing them, adding new elements, and so on.


// Gets the value of the alt attribute
$('img').attr('alt');

// Sets the value of the alt attribute
$('img').attr('alt', 'Wild kangaroo');

The above code shows you how to get and set the alt attribute of the image HTML tag.

Changing class names

One great way to manipulate how an element appears on the screen is to remove or add a CSS class to it.

				
$('ul li').addClass('list-item');
$('p').removeClass('highlight');
$('button').toggleClass('active');

The first one adds a CSS class .list-item to all li elements in a ul

The second one removes the CSS class .highlight from all p tags in the HTML

The third one checks if the button has the CSS class .active, then it removes it. If it doesn't have it, it adds that class

Updating CSS Styles with jQuery

The .css() method can handle any CSS changes in jQuery.

				
$('h1').css('font-size', '30px');

$('#header').css({
	height: '300px', 
	background: '#333'
});

The first line of code above shows you how to update a single CSS property, in this case we're updating the h1 tag and changing the font-size to 30px.

The second section shows you how to update multiple CSS properties at once.

Events

One of the most commonly used events is the click event. The click event is a method that gets triggered whenever someone clicks on the HTML element that is declared at the beginning of the click event method.


$('button').click(function(event){
  $('#header').css('background-color', 'blue');
});

In the above jQuery, we are creating an event that will go off whenever clicks on a button HTML element. When they click on it, this method will go after the element with an ID of header and change it's background color to blue.

jQuery Plugins

Because jQuery is widely used, a lot of people have created jQuery code(plugins) that you can download and easily add to your own site.

jQuery plugins will come with an external file that you will need to download and reference in your site. Then you 'initialize' their jQuery by connecting it to one of your own HTML elements.

Real world example: Slideshow

Now we're going to step through together how to add a slideshow jQuery plugin.

  • Start by Googling 'Slick Slider' or go to: http://kenwheeler.github.io/slick/
  • Click the 'Get it Now' menu item and then click the 'Download Now' button.
  • Unzip the slick-1.6.0 folder. Then open that folder and go inside the slick folder.
  • Copy the slick.min.js, slick.css, slick-theme.css, and entire fonts folder, and paste them into your project's folder.

Add the file references to your HTML

In the head tag section of your HTML, add these 2 stylesheets:

				
<link rel="stylesheet" type="text/css" href="slick.css" />
<link rel="stylesheet" type="text/css" href="slick-theme.css" />

AFTER your jQuery script tag, but BEFORE your main.js script tag, add the following:

				
<script src="slick.min.js"></script>

Setup your HTML markup

In your HTML, add the following HTML where you want your slider to appear.


<div class="slideshow">
  <div>your content</div>
  <div>your content</div>
  <div>your content</div>
</div>

Add in your slider images

Download 3 images from the internet and put them in your project folder.

Where it says 'your content' in the HTML, add 3 img tags.


<div class="slideshow">
  <div><img src="cupcake.png" /></div>
  <div><img src="frosting.png" /></div>
  <div><img src="cake.png" /></div>
</div>

Initialize your jQuery slider

In your main.js file, add the following code: 

			  
$('.slideshow').slick({
	infinite: true,
	slidesToShow: 1,
	slidesToScroll: 1,
	speed: 300,
	autoplay: true,
	fade: true,
	cssEase: 'linear'
});
